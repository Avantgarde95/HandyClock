# HandyClock

Simple clock written in python

### Requirements
- Python 2.X (Recommended : 2.7 / Run `.py` file.)
- or Python 3.X (Run `.py3` file.)
- Tkinter should be enabled.

### How to run
Run ```python HandyClock.py```.
